## Cara Instalasi

### Prasyarat
- Flutter SDK
- Dart SDK
- Android Studio / VS Code

### Langkah Instalasi
1. Clone repository
```bash
git clone https://github.com/Zivalez/LoveCoffeeApp.git
```
2. Pindah folder
```bash
cd LoveCoffeeApp
```

3. Install dependencies
```bash
flutter pub get
```

4. Jalankan aplikasi
```bash
flutter run
```
